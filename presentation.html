Кинотавр

Умный Telegram бот для подбора фильмов по настроению с современным Mini App интерфейсом

О проекте

Кинотавр — это инновационный Telegram бот, который помогает пользователям находить идеальный фильм для просмотра, основываясь на их текущем настроении и предпочтениях. Бот использует интеллектуальный диалоговый интерфейс для уточнения предпочтений и предлагает персонализированные рекомендации.

Ключевые возможности

Подбор по настроению

Интеллектуальная система анализа настроения пользователя через естественный диалог для точного подбора фильма

Бот задает уточняющие вопросы, чтобы лучше понять ваши предпочтения и найти идеальное совпадение

Архитектура проекта

Продуманная структура для масштабируемости и поддержки

🤖

Telegram Bot

Aiogram 3 для обработки сообщений, команд и взаимодействия с пользователями

🌐

Web Server

Aiohttp для обслуживания Mini App и статических файлов

🔌

Backend API

Интеграция с FastAPI сервисом для обработки диалогов и рекомендаций

💾

Хранение данных

Session-based хранилище для диалогов, LocalStorage для истории клиента

🎨

Frontend

Vanilla JavaScript с Telegram WebApp SDK для нативной интеграции

🚀

Deployment

Готов к развертыванию на любом хостинге с поддержкой Python
